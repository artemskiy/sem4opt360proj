# OptiBench

Python-библиотека для бенчмаркинга и визуализации сходимости стохастических оптимизаторов.

## Структура проекта
- `optibench/` — ядро библиотеки (запуск тестов, целевые функции, сбор логов).
- `bench_basic.py` — скрипт-точка входа для запуска базового прототипа.
- `config.yaml` — декларативное описание экспериментов.

## Быстрый старт
1. Создайте окружение из конфигурации:
   ```bash
   conda env create -f environment.yml
   conda activate optibench
2. Запустите базовый тест:
   ```bash
   python bench_basic.py
